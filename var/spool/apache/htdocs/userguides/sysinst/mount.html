• Now That The Disks Are Partioned And The Partitions Formatted You Can Now Allocate These Partitions To The Directory Structure
  • The Directory Structure Was Decided On When The Disks Were Partitioned Now We Need To Tell The System How You Intend To Map Them
  • You Need To Start From The Root (/) And Then Build It Up Until The Tree Is Completed
  • Always Put Lower Directories On First IE /var Must Be Defined Before /var/home
  • When All The Required Partitions Are Defined As Mount Points You Must Mount The Directory Tree To Begin Installing The Software
  • Select The Partition To Mount
  • Type In The Path For This Partition
  • If All Partitions Are Allocated You Will Be Taken Back To The Main Menu
  • Select The Mount Partitions Option From The Main Menu
  • Once The Mount Process Is Completed A List Of All Active Mounts Is Displayed
  • The New System Is Mounted Rooted To /mnt Ready For Installation
