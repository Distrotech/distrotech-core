• The Default Password On The System Is install This Is Not Secure And Should Be Changed (When Sentinel Solutions Firewall Is Installed Remote Access Is Possible) This Password Should Be Unguessable And Contain Non Alpha Numeric Characters IE !@#$%^&*()
