• When Network Sentinel Solutions Firewall Boots From The CD It Will Prompt You As To Whether You Will Be Running The Complete Package Or A Simple Gateway
  • If You Booting From The Gateway CD You Will Not Be Prompted
  • The IP Address Of The Server Must Be Setup This Is The Internal Address
  • You Should Only Use An Address In The Following Ranges And Subnet Bits Shown
    • 10.0.0.0 - 10.255.255.255 8
    • 172.16.0.0 - 172.16.31.255 16
    • 192.168.0.0 - 192.168.255.255 24
  • The Network Address Is The Lowest Address In A Subnet
  • The Network Address And Broadcast Address (Highest Address) Cannot Be Used As IP Address
  • The Following Table Outlines The Network Addresses As Used Above And With The Subnet Bits Used As Above. Replace X With The Value Chosen For That Position
    • 10.0.0.0 - 10.255.255.255 8 x.0.0.0
    • 172.16.0.0 - 172.31.255.255 16 x.x.0.0
    • 192.168.0.0 - 192.168.255.255 24 x.x.x.0
  • The Subnet Bits Represent The Size Of The Network The Lower The Bits The More IP's Are Available
  • When Using A Non Standard Mask Remember To Supply The Correct Network Address Above
  • Here Are The Most Common Subnet Bits Masks And Network Sizes
    • 8 255.0.0.0 16 Million (Class A Network) Used From 0.0.0.0 To 127.255.255.255
    • 16 255.255.0.0 65535 (Class B Network) Used From 128.0.0.0 To 191.255.255.255
    • 24 255.255.255.0 256 (Class C Network) Used From 192.0.0.0 To 223.255.255.255
  • If You Are Installing The Full Network Sentinel Solutions Firewall You May Create A Initial BOOTP Record To Enable A Machine On The Network To Obtain Network Settings Automatically
  • This Address Must Be In The Network Space You Have Allocated Above
  • You Require The Media Access Controller Address Of The Machine
    • To Obtain The MAC Address Of A Windows 95/98/Me System Click On Start Click On Run Then Type In winipcfg
    • To Obtain The MAC Address Of A Windows XP/2000 System Open Up Command Prompt And Type ipconfig /all
  • Enter The MAC Address You Obtained Above
  • The PC Whose MAC Address Has Been Entered Here Needs To Be Configured To Use DHCP See The PC Configuration Guide On This Site For More Information
  • All Other DHCP/BOOTP Servers Must Be Turned Off Before Proceeding
  • Once The New Firewall Has Been Configured And Is Running Reboot The PC You Have Configured As The Manager To Access And Configure The System
