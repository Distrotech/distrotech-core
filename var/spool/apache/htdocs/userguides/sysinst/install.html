• When You Are Ready To Install The System Select Install From The Main Menu
  • The Files Will Begin To Be Copied And A Progress Bar Displayed
  • Once The Files Are On The Computer It Needs To Prepare The System To Be Bootable
  • A List Of All Disks Is Displayed Please Select The Disk That Contains The Bootable Partition (/ Or /boot)
  • Network Sentinel Solutions Firewall Supports Both SMP An Non SMP Motherboards (Multiple Processors) You Need To Select The Option Applicable To You
  • A SMP Motherboard May Need To Be Configured Non SMP If Only One CPU Is Onboard
  • The Hard Drive Addressing Scheme Is Important As If Not Set Right The System May Need To Be Reinstalled
  • Linear Is Relatively Failsafe On Small Disks And Older Systems Especially If A /boot Partition Is Created Close To The Beginning Of The Disk
  • The System Needs To Configure The Boot Manager And Some Systems Before It Is Used For The First Time This Does Take Time Generally 2/3 Minutes On Faster Machines
  • If You Have Got To This Stage You Are Ready To Reboot And Use Your System
  • Don't Forget To Change The Password
