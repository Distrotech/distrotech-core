• To Begin Setting Up Internet Explorer Proxy Settings Click On The Start Bar
  • Select Settings
  • Select Control Panel
  • Double Click On The Internet Options Icon
  • Select The Connections Tab
  • Click The Lan Settings Button
  • Ensure Only "Use Proxy Server For Your Lan" Is Selected
  • Fill In The Name Of Your Networksentry Firewall (Hostname Configured On The Firewall)
  • Fill In The Port As 3128
  • Click On The Advanced Button
  • Select Use Same Proxy Server For All Protocols
  • The Proxy Address Should Be Set As In Previous Step To The Firewall Hostname
  • Click Ok
  • Click Ok Again To Exit The Internet Options Control Panel
