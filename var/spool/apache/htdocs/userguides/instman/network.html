• To Begin Setting Up Your Network Click On The Start Bar
  • Select Settings
  • Select Control Panel
  • Select Network By Double Clicking The Network Icon
  • For Your System To Work You Require At Least The Following Items
    • Client For Microsoft Networks
    • A Network Card In This Case A Realtek 8139
    • TCP/IP Set Of Protocols
    • File And Printer Shareing if you have files or a printer you want to share
  • To Log On To The Network Sentinel Solutions Firewall Select Client For Microsoft Networks As Your Default Login
  • Click On File And Printer Shareing To Enable/Disable As Required
  • File And Printer Shareing Exposes Your System To Posible Security Breaches Within Yuor Network
  • Click On The Identification Tab At The Top Of The Control Panel
  • For You To Become Part Of The Network You Need To Configure A Machine Identity
  • Enter The Name You Want This Computer To Be Known As On The Network IE Your Name
  • Enter The Workgroup You Are Joining This Could Be Your Division IE SALES.This Is Also The Group Shown When You Browse The Network.It Is Wise To Plan Workgroups In Advance.In A Small Company The Work Group Could Be The Same For All Users.
  • Click On Configuration On The Network Control Panel
  • Highlight Client For Microsoft Networks
  • Click On The Properties Button
  • Select Log On To Windows NT Domain
  • In The Space Provided Type In The Domain The Firewall Has Been Configured With
  • It Is Recomended That You Also Select Logon And Restore Network Connections
  • When You Finished Click Ok To Go Back To The Network Control Pannel
  • Click On TCP/IP (Sometimes More Than One Is Present Select The One Assosiated With Your Network Card)
  • Click On Properties
  • Click On The IP Address Tab
  • Enable Obtain IP Address Automaticaly (If DHCP Is Enabled On Firewall This Is The Default/Recomended) Otherwise Enter A IP Address.
  • IF DHCP Is Not Enabled You Need To Supply The Appropriate Netmask
  • Click On The WINS Configuration Tab
  • If DHCP IS Enabled (See Above) Select Use DHCP For WINS Resolution
  • If DHCP Is Disabled Apply The Appropriate Settings
  • CLick On The Gateway Tab
  • If DHCP Is Used This Must Be Left Unset Otherwise The Appropriate IP Must Be Set (This Is Normaly The Address Of The Network Sentinel Solutions Firewall
  • Click On The DNS Configuration Tab
  • This Must Be Set To Disable When Useing DHCP
  • If DHCP Is Not Used The Computer Name (See Above) Must Be Put In The Host Field The Internet Domain Name Must Be Put Into The Domain Field.At Least One DNS Server Must Be Specified (The Network Sentinel Solutions Firewall Is A DNS Server) And The Domain Must Be Added To The Domain Suffix Search Order
  • Click On The NetBios Tab
  • This Should Be Enabled To Be Able To Access File Servers Useing The TCP/IP Protocol
  • Click On Advanced
  • The TCP/IP Protocol Attached To Your Network Should Genraly Be The Default Protocol
  • Contact Your Network Administrator For More Information This In Most Cases Will Never Be Changed.
  • Click On The Bindings Tab
  • The Default Settings Are Almost Always Appropriate Only Change If You Are Required To By Your Network Administrator
  • Click On The Ok Button To Take You Back To The Network Control Pannel
  • Press OK
  • Reboot Your PC When Prompted If You Are Not Prompted Please Reboot Manualy
  • When You Log Back In Open The Network Control Pannel As Explaind In The First 3 Steps Above
  • Click On The Access Control Tab In The Network Control Pannel
  • Select User Level Access Control
  • The Domain Set On The Firewall Should Appear In The Domain Box
  • Click OK
  • Click Ok Again
  • You Will Be Required To Reboot Your System Again This May Be Delayed Till After Configureing User Profiles However It Is Recomended To Reboot At This Point
