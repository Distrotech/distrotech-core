Windows XP On A Network Sentinel Solutions Firewall

  • XP Home Cannot Be Used To Access A Network Sentinel Solutions Firewall
  • Useing The Network ID Wizzard You Need To Do The Following
    • Supply A Username And Password For A User On The Domain
    • Supply The Domain For The PC To Join
    • Only A Admistrator On The Network Sentinel Solutions Firewall Can Join The PC To The Domain
    • When Asked To Add A User Select "Not At This Time"
  • The System Needs To Be Restarted
  • Login As A Local Administrator And Add The User For The Domain
  • Logoff And Logon As The User You Have Added
  • Follow The Guide Lines For Configureing Proxy,Email And Ldap Settings As For Windows 9x/Me
