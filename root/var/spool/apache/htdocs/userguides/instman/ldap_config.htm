• Open Outlook Click On Tools Then Accounts
  • Click On Add
  • Select The Directory Service Option
  • You Need To Add A Different "Account" For Each Address Book You Need Access To
  • Enter The Address Of Your Firewall The Full Name Hostname And Doamin Ie firewall.company.co.za
  • You Will Need To Logon To Protect Access From Unauthorised Users On The Internet,Tick My LDAP Server Requires Logon
  • Your Username Must Be In The Form Of A LDAP DN There Are Two Posibilities
    • I Need Access To Address Books With Read Access For All Users.Your Login Will Be uid=<USERNAME>,ou=user
    • I Have Access To A Closed Address Book.Your Login Will Be cn=<USERNAME>,ou=users,cn=<ADDRESSBOOK>.
  • Do Not Select Secure Password Authentication
  • Select Yes To Enable This Feature (Recomended)
  • Click Finish To Continue
  • Locate The Account You Have Just Created And Click The Properties Button
  • Click On The Advanced Tab And Select "This Server Requires A Secure Connection"
  • The Search Base Is The Begining Point Of The Address Book You Are Configureing For The Search Base Is In The Form Of A LDAP DN ou=entries,cn=<ADDRESSBOOK>
  • The Search Base ou=users is the base of all configured users (those with email accounts on the server).
